These are .rbz or .apk files. Technically they are zip archives, but you don't need to extract them!

2. Install the extension

For PC:
  • Open Sketchup
  • In the top menu bar click on "Window" then click on "Extension Manager"
  • In the bottom left click "Install"
  • Navigate to where you downloaded the ".rbz" file. Select it and click "Install"
  • Done!

If an older VR Sketch was already installed, you need to restart SketchUp now.

For Oculus Quest: see this page.

For Pico Neo 3: see this page.

3. Have fun

VR Sketch is now installed, just click "Extension", "VR Sketch", "Send to VR" to start working in VR!

For more info see the Documentation.
